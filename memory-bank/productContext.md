# Product Context

## Цель проекта
Создание удобного и эстетичного веб-приложения для чтения документов (Zen Reader). Приложение должно обеспечивать комфортный пользовательский опыт ("zen"), вероятно, с минималистичным интерфейсом.

## Текущее состояние
Проект находится на стадии инициализации.
Существует базовая структура на React + Vite.
Установлены зависимости для работы с PDF (`react-pdf`) и управления состоянием (`zustand`).

## Ключевые функции (предполагаемые)
- Загрузка и просмотр PDF документов.
- Навигация по страницам.
- Масштабирование.
- Возможно, сохранение прогресса чтения (local storage / store).

## Проблемы и вызовы
- Настройка корректного отображения PDF (pdf worker).
- Оптимизация производительности при рендеринге больших документов.
- Обеспечение адаптивности.
