# Tech Context

## Стек технологий
- **Язык**: TypeScript (`~5.8.2`)
- **UI Framework**: React (`^19.2.3`)
- **Сборщик**: Vite (`^6.2.0`)
- **Стейт-менеджмент**: Zustand (`^5.0.9`)
- **Работа с документами**: react-pdf (`^10.2.0`)
- **Иконки**: lucide-react (`^0.561.0`)
- **Утилиты**: uuid (`^13.0.0`)

## Среда разработки
- **OS**: Windows
- **Node.js**: (Версия пользователя, использует `@types/node ^22.14.0`)
- **Браузер**: BrowserMCP (для тестов) / Chrome (основной).

## Конфигурация
- **TypeScript**: `tsconfig.json` (стандартный Vite пресет).
- **Vite**: `vite.config.ts`.
